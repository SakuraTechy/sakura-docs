# Sakura Docs

## 介绍

🔥Sakura Docs🔥是自动化平台官方文档，采用 VitePress 技术构建，专注于打造快速、现代化且以内容为核心的文档站点，兼具高效开发体验与优雅展示效果。全面介绍了自动化平台的基础功能、核心功能及产品优势，涵盖系统管理、项目管理，测试管理，接口管理，自动化管理等模块，突出其在提升研发效率与优化用户体验方面的卓越能力，展现了一站式持续自动化平台的强大实力与专业价值。
